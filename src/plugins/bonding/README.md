# NAME

**ncollectd-bonding** - Documentation of ncollectd's bonding plugin

# SYNOPSIS

	load-plugin bonding
	plugin bonding

# DESCRIPTION

The **bonding** plugin reports the number of slave interfaces on
a bonding device and the number of those interfaces that are active.
