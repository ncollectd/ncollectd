NCOLLECTD-RTCACHE(5) - File Formats Manual

# NAME

**ncollectd-rtcache** - Documentation of ncollectd's rtcache plugin

# SYNOPSIS

	load-plugin rtcache
	plugin rtcache

# DESCRIPTION

The **rtcache** plugin collect routing cache statistics
from the file */proc/net/stat/rt\_cache*.
