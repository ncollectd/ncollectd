NCOLLECTD-MEMORY(5) - File Formats Manual

# NAME

**ncollectd-memory** - Documentation of ncollectd's memory plugin

# SYNOPSIS

	load-plugin memory
	plugin memory

# DESCRIPTION

The **memory** plugin reports the free memory and different kinds
of used memory in the system.
