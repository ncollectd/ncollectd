NCOLLECTD-LOCKS(5) - File Formats Manual

# NAME

**ncollectd-locks** - Documentation of ncollectd's locks plugin

# SYNOPSIS

	load-plugin locks
	plugin locks

# DESCRIPTION

The **locks** plugin collect the file locks and locks pending
from */proc/locks*.
