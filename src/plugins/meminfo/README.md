# NAME

**ncollectd-meminfo** - Documentation of ncollectd's meminfo plugin

# SYNOPSIS

	load-plugin meminfo
	plugin meminfo

# DESCRIPTION

The **meminfo** plugin reports the metrics from the */proc/meminfo* file.
