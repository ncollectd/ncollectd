# NAME

**ncollectd-wpar** - Documentation of ncollectd's wpar plugin

# SYNOPSIS

	load-plugin wpar
	plugin wpar

# DESCRIPTION

The **wpar** plugin reads statistics from the *Workload Partitions*
running in the system.
