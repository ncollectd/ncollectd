# NAME

**ncollectd-serial** - Documentation of ncollectd's serial plugin

# SYNOPSIS

	load-plugin serial
	plugin serial

# DESCRIPTION

The **serial** plugin collects statistics of the traffic on serial interfaces,
reading the file /proc/tty/driver/serial.
