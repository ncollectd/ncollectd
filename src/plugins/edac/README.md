# NAME

**ncollectd-edac** - Documentation of ncollectd's edac plugin

# SYNOPSIS

	load-plugin edac
	plugin edac

# DESCRIPTION

The **edac** plugin collect report correctable and uncorrectable memory
errors from the Error Detection And Correction (EDAC) kernel subsystem.
