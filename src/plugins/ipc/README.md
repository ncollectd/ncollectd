# NAME

**ncollectd-ipc** - Documentation of ncollectd's ipc plugin

# SYNOPSIS

	load-plugin ipc
	plugin ipc

# DESCRIPTION

The **ipc** plugin collect metrics from the System V IPC about messages,
semaphores, and shared memory.
