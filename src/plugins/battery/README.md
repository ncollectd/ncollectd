# NAME

**ncollectd-battery** - Documentation of ncollectd's battery plugin

# SYNOPSIS

	load-plugin battery
	plugin battery

# DESCRIPTION

The **battery** plugin reports the remaining capacity, power and voltage of
laptop batteries.
