# NAME

**ncollectd-drbd** - Documentation of ncollectd's drbd plugin

# SYNOPSIS

	load-plugin drbd
	plugin drbd

# DESCRIPTION

The **drbd** plugin collect statistics from the Distributed Replicated
Block Device (DRBD) reading the file */proc/drbd*.
