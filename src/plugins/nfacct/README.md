# NAME

**ncollectd-nfacct** - Documentation of ncollectd's nfacct plugin

# SYNOPSIS

	load-plugin nfacct
	plugin nfacct

# DESCRIPTION

The **nfacct** plugin retrieve netfilter accounting objects.
With the command nfacct you can create/retrieve/delete
this accounting objects.
