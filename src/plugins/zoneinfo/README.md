# NAME

**ncollectd-zoneinfo** - Documentation of ncollectd's zoneinfo plugin

# SYNOPSIS

	load-plugin zoneinfo
	plugin zoneinfo

# DESCRIPTION

The **zoneinfo** plugin collectd information about memory zones reading
the file */proc/zoneinfo*.
