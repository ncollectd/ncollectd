# NAME

**ncollectd-arp** - Documentation of ncollectd's arp plugin

# SYNOPSIS

	load-plugin arp
	plugin arp

# DESCRIPTION

The **arp** plugin counts the number or entries in the kernel ARP table per
interface, reading the file */proc/net/arp*.
