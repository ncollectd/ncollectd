# NAME

**ncollectd-slabinfo** - Documentation of ncollectd's slabinfo plugin

# SYNOPSIS

	load-plugin slabinfo
	plugin slabinfo

# DESCRIPTION

The **slabinfo** plugin collect information about kernel caches reading the
file /proc/slabinfo.
