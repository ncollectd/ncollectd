# NAME

**ncollectd-lvm** - Documentation of ncollectd's lvm plugin

# SYNOPSIS

	load-plugin lvm
	plugin lvm

# DESCRIPTION

The **lvm** plugin collects metrics form the Logical Volume Manager (LVM),
of the usage of Volume Grousp (VG) and Logical Volumes (LV).
