# NAME

**ncollectd-schedstat** - Documentation of ncollectd's schedstat plugin

# SYNOPSIS

	load-plugin schedstat
	plugin schedstat

# DESCRIPTION

The **schedstat** plugin collectd scheduling metrics from
*/proc/schedstat*.
