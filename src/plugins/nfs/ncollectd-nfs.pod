=head1 NAME

ncollectd-nfs - Documentation of ncollectd's C<nfs plugin>

=head2 Plugin C<nfs>

The I<nfs plugin> collects information about the client usage of the Network
File System (NFS). It counts the number of procedure calls for each procedure,
grouped by version.
